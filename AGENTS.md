# Repository Instructions

- Use English for all source code, identifiers, comments, tests, documentation, user-facing copy, commit messages, issues, and pull requests.
- Keep `README.md` in English.
- Implement Zuwerk independently. Do not copy code, text, visual design, or implementation details from restricted-license reference products.
- Keep the CLI small, scriptable, and friendly to both humans and agents.
- Prefer structured JSON for machine-readable output and verify behavior with automated tests.
